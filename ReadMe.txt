========================================================================
    アクティブ テンプレート ライブラリ : CopyPathX プロジェクトの概要
========================================================================

パスコピーツール

Windows 上でエクスプローラの右クリックでパスをコピーするツールです。

□使い方
-------------------------------------------------------------------
エクスプローラからファイルを右クリックすることでクリップボードに
パスをコピーします。
コンテキストメニューから[パスをコピー]を選びます。
Shiftキーを押しながら実行するとショートパスになり、CtrlキーでC言語
スタイルのパスをコピーしています。

また、複数のファイルを選択した場合は改行で区切ってコピーします。

機能を選択するダイアログも表示できます。
  ダイアログの表示方法(マウス左右入れ替え時は逆の操作)：
	1.右クリックでメニュー表示
	2.マウスの左を押しながら右クリックでメニュー選択

□サンプル:
	・デフォルト
		C:\Program Files\desktop.ini
	・ショートパス(Shift)
		C:\PROGRA~1\desktop.ini
	・Cスタイル(左Ctrl)*
		C:\\Program Files\\desktop.ini
	・スラッシュスタイル(右Ctrl)*
		C:/Program Files/desktop.ini
	・ショートパス + Cスタイル(Shift + 左Ctrl)*
		C:\\PROGRA~1\\desktop.ini


□インストール
-------------------------------------------------------------------
適当なフォルダに展開し、Setup.exeを実行して「はい」を選んでください。
レジストリに登録されます。

*旧バージョンから更新する場合は、Setup.exeを実行して「いいえ」を選んで、
いったんレジストリから削除してください。
その後、DLLファイルなどをコピーし再度Setup.exeを実行し登録します。
option.iniファイルは必要に応じて書き換えてください。

□アンインストール
-------------------------------------------------------------------
Setup.exeを実行して「いいえ」を選んでください。
レジストリから削除されます。
ファイルを削除する場合は、いったんログオフか再起動を行って削除してください。

□設定
-------------------------------------------------------------------
option.iniファイルを読み込んで機能を変更できます。

・addbackslash
	0 = ディレクトリ名の最後に\(/)を追加します	(デフォルト)
	1 = 1を指定すると追加しません
		サンプル:
			C:\Windows

・ctrlstyle
	Windows9x、Meでは判別できないのでどちら一方の機能を選択します。
	NT系OS(NT4.0/2000/XP/2003)は無視します。
	0 = スラッシュスタイル(/)
	1 = Cスタイル(\\)	(デフォルト)

・menutext
	メニューに表示する文字を設定します。
	例：パスをコピー(&C)
	&の後にアルファベットを指定することでアクセスキーも指定可能。
	
	
	
□開発環境:
-------------------------------------------------------------------
Visual Studio 2010



-------------------------------------------------------------------
-------------------------------------------------------------------

AppWizard が作成したこの CopyPathX アプリケーションには、
ダイナミック リンク ライブラリ (DLL) の基本的な使い方が示されています。アプリケーション作成のひな型としてお使いください。
このプロジェクトは Visual C++ 属性を使用して実装されました。

このファイルにはプロジェクトを構成している各ファイルの
概要説明が含まれています。

CopyPathX.vcproj
    これはアプリケーション ウィザードで生成された VC++ プロジェクトのメイン プロジェクト ファイルです。
    ファイルが生成された Visual C++ のバージョン情報が含まれています。 
    また、アプリケーション ウィザードで選択したプラットフォーム、構成およびプロジェクト機能に関する
    情報も含まれています。

_CopyPathX.idl
    このファイルはプロジェクトがビルドされるときにコンパイラで生成されます。タイプ ライブラリの IDL 定義、 
    プロジェクトで定義されたインターフェイス、およびコクラスを含んでいます。
    このファイルは MIDL コンパイラによって処理され、以下のファイルを生成します :
        C++ インターフェイス定義および GUID 宣言 (_CopyPathX.h)
        GUID 宣言                                (_CopyPathX_i.c)
        タイプ ライブラリ                                  (_CopyPathX.tlb)
        マーシャリング コード                                 (_CopyPathX_p.c and dlldata.c)
CopyPathX.cpp
    このファイルはオブジェクト マップおよび DLL エクスポートの実装を含んでいます。
CopyPathX.rc
    これはプログラムが使用する Microsoft Windows のリソースの
    一覧ファイルです。

CopyPathX.def
    このモジュール定義ファイルは、DLL で必要なエクスポートに関する情報へのリンカを提供し、
    次のエクスポート情報を含んでいます :
        DllGetClassObject  
        DllCanUnloadNow    
        GetProxyDllInfo    
        DllRegisterServer	
        DllUnregisterServer

その他の標準ファイル :

StdAfx.h, StdAfx.cpp
    これらのファイルはプリコンパイル済み (PCH) ヘッダー ファイル CopyPathX.pch、
    およびプリコンパイルされた型のファイル StdAfx.obj をビルドするために使われます。

Resource.h
    このファイルはリソース ID を定義する標準ヘッダー ファイルです。

/////////////////////////////////////////////////////////////////////////////
プロキシ/スタブ DLL プロジェクトおよびモジュール定義ファイル :

CopyPathXps.vcproj
    このファイルは必要に応じてプロキシ/スタブのビルドに使用されるプロジェクト ファイルです。
	主なプロジェクトの IDL ファイルには少なくともインターフェイスを 1 つ含み、
	プロキシ/スタブ DLL をビルドする前に IDL ファイルをコンパイルする必要があります。	この過程で
\プロキシ/スタブ DLL をビルドするのに必要な tdlldata.c、CopyPathX_i.c および CopyPathX_p.c が
	生成されます。

CopyPathXps.def
    このモジュール定義ファイルは、プロキシ/スタブで必要なエクスポートに関する
    情報へのリンカを提供します。
